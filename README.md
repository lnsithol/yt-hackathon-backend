# YouTube Transcript Analysis API

This Flask application provides an API for analyzing YouTube video transcripts. It offers various endpoints for text analysis, including word frequency, sentiment analysis, named entity recognition, and topic modeling.

## Features

- Word frequency analysis
- Sentiment distribution
- Named entity recognition
- Average sentiment calculation
- Topic modeling
- Retrieval of all transcripts

## Prerequisites

Before running this application, make sure you have the following installed:

- Python 3.7+
- Flask
- pandas
- numpy
- nltk
- scikit-learn
- spacy
- flask-cors
